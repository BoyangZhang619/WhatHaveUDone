# 📚 Study Log — whatIveDone

一个简洁的在线学习记录工具，帮你随时记录每天学了什么、学了多久、专注如何，方便日后回顾与复盘。

## 技术栈

- **前端**：Vue 3 + TypeScript（Composition API）
- **构建**：Vite 7
- **国际化**：vue-i18n（中文简体 / 繁体、English）
- **后端**：Cloudflare Workers + D1（SQLite）
- **主题**：CSS 变量驱动的亮色 / 暗色双主题

## 功能一览

### 🔐 登录与注册

- 打开网站后进入登录页面，页面会自动获取并显示你的 IP 地址。
- 输入邮箱和密码即可**注册**新账户或**登录**已有账户。
- 登录成功后自动跳转到主界面；支持安全登出。
- 登录页底部可快速切换界面语言（🌐）。

### 📝 新增学习记录

- 在主界面点击「**新增学习记录**」卡片，会打开全屏编辑器。
- 可填写以下信息：
  - **标题** — 简要描述本次学习内容（最多 120 字）
  - **学习时间** — 记录具体的开始时间
  - **时长** — 本次学习花了多少分钟
  - **专注度 / 难度** — 通过滑块打分（1–5）
  - **标签** — 按回车添加自定义标签，也可点击快捷标签（reading / coding / notes / review）
  - **目标** — 这次学习想解决什么问题
  - **详述** — 记录做了什么、学到了什么、卡在哪里、下一步计划（支持预览模式）
  - **下一步行动（Todos）** — 添加待办事项，可勾选完成状态
  - **置顶** — 可标记重要记录为置顶
- 提供 7 种模板快速填充：背诵、阅读、编码、作业、任务/项目、课程/讲座、复盘/总结。
- 支持**草稿自动保存**到本地（可在设置中开关），刷新页面不会丢失。
- 实时显示草稿状态和字数统计。
- 快捷键：`Ctrl / ⌘ + Enter` 快速保存，`Esc` 关闭编辑器。
- 数据存储在 Cloudflare D1（SQLite）中。

### 📋 浏览与管理记录

- 主界面以**卡片网格**形式展示所有学习记录，置顶记录带有高亮标记。
- 每张卡片显示标题、时间、时长、专注度、难度、更新时间、内容预览、目标、待办事项和标签。
- 点击「**详情**」可查看完整记录，包含：
  - 内容全文与字数统计
  - 专注度 / 难度可视化仪表盘
  - 时长进度条（基于 180 分钟）
  - 目标与元数据一览
  - 待办事项列表（含完成状态）
- 支持**删除记录**（可开启二次确认）。
- 支持**分页浏览**，智能页码：≤4 页全显示，>4 页显示首尾页 + 当前页 + 省略号。
- 支持页码跳转和总数显示。
- 支持多种排序方式：置顶优先、最新优先、最早优先、时长最长。
- 点击「**刷新**」可重新加载最新数据。

### ⚙️ 设置

设置页面分为五个分组，所有通用与外观设置保存在本地：

#### 通用

- **语言** — 中文简体、中文繁体、English 三语切换
- **每页默认条数** — 1 / 3 / 5 / 10 / 15 / 20 / 25 / 50
- **默认排序方式** — 置顶优先 / 最新 / 最早 / 时长最长
- **草稿自动保存** — 开关控制
- **删除前确认** — 开关控制
- **键盘快捷键** — 开关控制
- 🔒 时区（开发中）

#### 外观

- **主题** — 亮色 / 暗色一键切换，完整的 CSS 变量暗色适配
- **字体大小** — 12px–20px 滑块调节，实时生效
- **界面动画** — 一键禁用所有过渡与动画，适合低性能设备
- **圆角风格** — 直角 / 小圆角 / 中等 / 大圆角（默认）
- **卡片预览行数** — 2 / 3 / 5 / 8 行或全部展开
- 🔒 卡片布局（开发中）

#### 账户

- **清空数据** — 全给你删了 嘻嘻
- **数据导出** — 自动下载json文件
- 🔒 昵称、密码、邮箱修改（开发中）
- 🔒 导入（开发中）
- 🔒 会话管理（开发中）
- 🔒 注销账户（开发中）

#### 通知与提醒

- 🔒 每日学习提醒、打卡提醒、每周周报（开发中）

#### 关于

- 版本号显示（v1.1.0）
- 检查更新
- 意见反馈（邮件）
- GitHub 源码链接
- **重置所有设置** — 一键恢复全部偏好为默认值

### 🎨 设计特点

- **黑灰白简约风** — 纯净的设计语言，无冗杂纹理和渐变
- **暗色模式** — 完整的 Dark Theme 支持，所有页面和表单控件适配
- **CSS 变量架构** — 30+ 设计 token 统一管理颜色、阴影、圆角
- **响应式布局** — 手机和电脑均可舒适使用
- **柔和过渡动画** — 页面切换、折叠展开等均有流畅动画（可全局关闭）
- **所有外观设置刷新后自动恢复**
